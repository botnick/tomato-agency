หลายคนรู้จัก **IDOL+** (idol plus) ในฐานะแอพไลฟ์สด แต่พอเข้าไปเล่นจริงจะเจอว่าในแอพมี **เกมให้เล่นในตัว** ด้วย ไม่ใช่แค่เปิดกล้องคุยอย่างเดียว ส่วนรวมเกมนี้เรียกว่า **Game Hub** และเกมพวกนี้ใช้เหรียญที่ชื่อ **Ruby** บทความนี้จะพาไปดูว่า Game Hub คืออะไร Ruby เอาไว้ทำอะไร และได้ Ruby มายังไงแบบเข้าใจง่าย

## IDOL+ ไม่ได้มีแค่ไลฟ์คุย

จุดขายอย่างหนึ่งของ IDOL+ (ชื่อเต็ม IDOL+ Global & Celeb Live) คือการรวม **ไลฟ์สด + เกม + ห้องปาร์ตี้** ไว้ในที่เดียว เพราะงั้นนอกจากไลฟ์โชว์ ร้อง เต้น พูดคุยแล้ว ในแอพยังมีโซนเกมให้เล่นเพื่อความสนุกด้วย

ถ้าเปิดหน้าโปรไฟล์ในแอพ จะเห็นเมนูฟีเจอร์หลายอันเรียงกัน เช่น Wallet, Skin, Avatar, Fan Club, Badges และก็มี **Game Hub** รวมอยู่ในนั้น นี่แหละคือประตูเข้าสู่โซนเกมของแอพ

## Game Hub คืออะไร

**Game Hub คือส่วนรวมเกมที่เล่นได้ภายในแอพ IDOL+** พูดง่าย ๆ คือมันเป็น "โซนเกม" ในแอพเดียวกับที่เราใช้ไลฟ์ ไม่ต้องออกไปโหลดเกมอื่น เล่นได้ในแอพเลย

ข้อดีของการมีเกมอยู่ในแอพไลฟ์คือมันเพิ่มลูกเล่นให้ทั้งคนดูและวีเจ — บางห้องใช้เกมเป็นกิจกรรมเรียกความสนุก สร้างจังหวะให้คนดูมีส่วนร่วม ไม่ใช่แค่นั่งดูเฉย ๆ

## Ruby คืออะไร เอาไว้ทำอะไร

เกมใน Game Hub ไม่ได้ใช้ Candy (ที่เอาไว้ส่งของขวัญ) แต่ใช้เหรียญอีกตัวชื่อ **Ruby**

**Ruby คือเหรียญสำหรับเล่นเกมภายในแอพโดยเฉพาะ** เป็นคนละสกุลกับ Candy และ Star ชัดเจน ในหน้ากระเป๋าเงิน (Wallet) ของแอพจะเห็น Ruby แยกเป็นอีกช่องหนึ่ง มีไอคอนเป็นเพชรสีชมพู ถ้าอยากเล่นเกมในแอพก็ต้องมี Ruby ในกระเป๋าก่อน

> ถ้ายังงงว่า Candy, Star, Ruby ต่างกันยังไง เราแยกอธิบายละเอียดไว้ที่ [Candy / Star / Ruby ต่างกันยังไง](/blog/idol-plus-candy-star-ruby/)

## เติม Ruby IDOL+ ยังไง? (จุดที่คนมักเข้าใจผิด)

เรื่องนี้มือใหม่พลาดกันเยอะ — **Ruby เติมตรงจากระบบไม่ได้** ต่างจาก Candy ที่เติมเข้ามาตรง ๆ ได้เลย ใครค้นหาว่า "เติม Ruby IDOL+ ยังไง" แล้วหาปุ่มเติม Ruby ตรง ๆ ไม่เจอ ไม่ต้องงง เพราะมันไม่มี

วิธีได้ Ruby มาคือต้อง **แลกจาก Candy** ในแอพ ขั้นตอนคร่าว ๆ:

1. **เติม Candy เข้ามาก่อน** — ผ่านแอพหรือร้านตัวแทน (อ่านวิธีที่ [วิธีเติมเงิน IDOL+](/blog/idol-plus-topup-guide/))
2. **เข้าหน้าแลกเหรียญ** — ในแอพจะมีหน้า Game Coins Exchange
3. **แลก Candy เป็น Ruby** — เลือกจำนวนที่ต้องการแล้วยืนยัน
4. **เอา Ruby ไปเล่นเกม** ใน Game Hub ได้เลย

พูดสั้น ๆ: **เติม Candy → แลกเป็น Ruby → เล่นเกม** นี่คือเส้นทางเดียวที่จะได้ Ruby มาใช้

## อัตราแลกดูได้ที่ไหน

**บทความนี้ตั้งใจไม่ระบุอัตราแลกแบบตายตัว** เพราะแอพปรับเปลี่ยนได้เป็นระยะ จำตัวเลขเก่ามาใช้อาจคลาดเคลื่อน ทางที่ชัวร์คือดูจากหน้าแลกเหรียญ (Game Coins Exchange) ในแอพ ณ ตอนนั้น — หน้านั้นจะโชว์อัตราปัจจุบันให้เห็นชัด ๆ ว่า Candy เท่านี้ได้ Ruby เท่าไหร่ ยึดตามที่แอพแสดงตอนแลกเสมอ

## เล่นเกมในแอพให้สนุกและไม่หลุดงบ

เกมในแอพเป็นลูกเล่นเสริมความสนุก แต่ก็ควรเล่นด้วยสติเหมือนทุกอย่างที่ใช้เงิน มีหลักง่าย ๆ:

- **ตั้งงบไว้ในใจ** — จะเล่นเกมก็ตั้งวงเงินที่รับได้ อย่าไหลไปเรื่อย ๆ
- **แยกให้ออกระหว่างซัพพอร์ตวีเจกับเล่นเกม** — Candy ส่งของขวัญกับ Ruby เล่นเกมคนละเรื่องกัน อย่าปนกัน
- **เติม/แลกผ่านช่องทางในแอพหรือร้านที่แอพรับรอง** — ปลอดภัยกว่าไปแลกกับคนแปลกหน้านอกระบบ
